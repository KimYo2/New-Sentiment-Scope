Panduan Penggunaan

Maksimalkan potensi SentimentScope dengan memahami setiap fiturnya. Pilih topik di bawah ini untuk mempelajari lebih lanjut.

Analisis Satuan

Deteksi sentimen secara instan untuk teks singkat seperti satu ulasan produk, caption, atau tweet.

Cara Pakai:

  1. Buka tab Analisis.
  2. Ketik teks di kolom input.
  3. Klik tombol Analisis Sekarang.

Analisis Batch

Hemat waktu dengan menganalisis ribuan data sekaligus menggunakan file CSV atau Excel.

Cara Pakai:

  1. Siapkan file .csv atau .xlsx.
  2. Pastikan ada kolom text / review.
  3. Upload di tab Batch & Analisis.

Media Intelligence

Pahami opini audiens Anda dengan menarik data komentar langsung dari video YouTube.

Cara Pakai:

  1. Salin URL video YouTube.
  2. Tempel di kolom input tab Media.
  3. Klik Analisis dan tunggu hasil.

Tips & Trik

Gunakan Bahasa Baku

Meskipun AI bisa memahami slang, bahasa baku memberikan hasil akurasi tertinggi hingga 99%.

Koreksi Manual

Jika hasil prediksi salah, Anda bisa mengoreksinya di tabel Riwayat (khusus user login).

Format Tanggal CSV

Untuk analisis batch, pastikan tidak ada karakter aneh di dalam file CSV Anda (gunakan UTF-8).

Perbarui Model Berkala

Lakukan training ulang setiap bulan jika trend bahasa atau topik pembicaraan berubah.

Tentang SentimentScope

Aplikasi analisis sentimen berbasis Deep Learning yang dirancang khusus untuk memahami nuansa Bahasa Indonesia. Dibangun dengan teknologi terkini untuk memberikan wawasan yang cepat, akurat, dan dapat diandalkan.

Teknologi AI

Didukung oleh IndoBERT (Bidirectional Encoder Representations from Transformers), model bahasa mutakhir yang telah dilatih pada jutaan kata Bahasa Indonesia untuk memahami konteks, slang, dan idiom lokal.

Fitur Unggulan

  • Real-time Analysis
  • Batch Processing (CSV/Excel)
  • Media Scraping
